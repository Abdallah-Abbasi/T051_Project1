تقسيم التردد (FDM)  مقابل  تقسيم الزمن (TDM)

تقسيم التردد (FDM)

  • تُقسَّم الترددات الكهرومغناطيسية إلى نطاقات ضيّقة.
  • يُخصَّص لكل اتصال نطاق مستقل ويمكنه الإرسال بأقصى معدل لذلك النطاق.

تقسيم الزمن (TDM)

  • يُقسَّم الزمن إلى خانات زمنية.
  • يُخصَّص لكل اتصال خانة دورية ويمكنه استخدام كامل النطاق فقط خلالها.

طبقات بروتوكول الإنترنت

  1. Application: تطبيقات الشبكة
    IMAP،‏ SMTP،‏ HTTP
  2. Transport: نقل بيانات بين العمليات
    TCP،‏ UDP
  3. Network: توجيه الحزم بين المصدر والوجهة
    IP،‏ بروتوكولات التوجيه
  4. Link: نقل البيانات بين عناصر الشبكة المتجاورة
    Ethernet،‏ Wi-Fi،‏ PPP
  5. Physical: نبضات على السلك
